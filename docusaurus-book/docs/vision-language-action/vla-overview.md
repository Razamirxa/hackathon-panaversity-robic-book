# VLA Model Architecture

VLA models use transformer architectures to combine modalities.

## Components

- **Vision Encoder**: Process images
- **Language Encoder**: Process text
- **Action Decoder**: Generate robot actions

## Summary

Transformers enable multimodal learning for robotics.
