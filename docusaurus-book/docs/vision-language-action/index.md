# Introduction to VLA

Vision-Language-Action models combine vision, language, and robot control.

## What are VLA Models?

VLA models process visual observations and language instructions to generate robot actions.

## Examples

- RT-1, RT-2 (Google)
- PaLM-E
- VIMA

## Summary

VLA models enable instruction-following robots.
