# Action Primitives

Define robot action spaces for VLA models.

## Discrete Actions

- Pick, place, push, pull
- Waypoint navigation

## Continuous Actions

- Joint positions/velocities
- End-effector poses

## Summary

Action primitives define how VLA models control robots.
