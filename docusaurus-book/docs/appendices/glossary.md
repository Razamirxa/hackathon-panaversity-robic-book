# Glossary

**Action**: Robot command or movement

**Digital Twin**: Virtual replica of physical robot

**DDS**: Data Distribution Service (ROS 2 middleware)

**Edge AI**: AI inference on robot hardware

**Gazebo**: Open-source robot simulator

**Isaac Sim**: NVIDIA photorealistic simulator

**Jetson**: NVIDIA edge AI platform

**LIDAR**: Light Detection and Ranging sensor

**Nav2**: ROS 2 navigation stack

**Odometry**: Robot pose estimation from wheel encoders

**ROS 2**: Robot Operating System 2

**SLAM**: Simultaneous Localization and Mapping

**URDF**: Unified Robot Description Format

**VLA**: Vision-Language-Action model

**ZMP**: Zero Moment Point (balance control)
